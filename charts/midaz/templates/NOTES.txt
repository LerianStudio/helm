Accessing midaz ledger services using port-forward in Helm

This documentation provides instructions on how to access the `midaz-ledger` services using port-forwarding in a Helm-managed Kubernetes environment.

Steps

1. Port-Forward to midaz-ledger:
    Use the following command to port-forward to the `midaz-ledger` service.

    kubectl port-forward svc/midaz-ledger 3002:3002

    This will forward port 3002 on your local machine to port 3002 on the `midaz-ledger` service.

Accessing the Services
Once the port-forwarding is set up, you can access the services in your web browser or via API calls using the following URLs:
- `midaz-ledger`: http://localhost:3002
